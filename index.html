3 Words or Less

How to Play

  1. The game selects 5 random words from the word bank.
  2. For each word, the player must give clues using THREE words or less.
  3. The host controls the game by clicking Correct, Pass, or Fault.
  4. Passed words will return later in the round.
  5. You have 2 minutes to guess all words. Try to finish before time runs out!

_

Game Setup for Host

  1. A tablet computer, laptop, or desktop should be ready and visible to the host and glue-giver.
  2. Word guessers are seated opposite the clue-giver.
